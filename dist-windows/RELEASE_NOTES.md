# Release Notes - SerialVision Pro v1.0.0

## New Features
✅ Direct serial communication via USB
✅ Real-time data plotting with Chart.js
✅ Hex viewer and spectrogram display
✅ Session persistence and export

## System Requirements
- Windows 10/11 (64-bit)
- 4GB RAM minimum
- 200MB free disk space
- USB port for serial devices

## Installation Instructions
1. Run `SerialVision-Pro-Setup.exe`
2. Follow the installation wizard
3. Launch from Start Menu

## Portable Version
Use `SerialVision-Pro-Portable.exe` for no-installation use.
